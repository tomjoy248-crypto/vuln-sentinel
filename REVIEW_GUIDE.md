# 评委评审指南

## 项目一句话

漏洞哨兵：面向中小企业的 AI 安全修复闭环平台。
扫描发现安全配置问题 → 生成 8 个修复入口配置 → 复测验证效果 → 导出 PDF 报告。
不是"又一个扫描器"，而是从发现到验证的完整闭环。

## 体验路径（7 步）

1. 打开 https://vuln-sentinel-v11-s.onrender.com，使用 demo / demo123 一键登录体验
2. 输入 example.com，点击"开始安全扫描"，3-5 秒出结果
3. 查看评分 + 风险等级，展开漏洞详情看修复建议
4. 查看报告页「修复前后价值对比」表格（评分、风险数、处理时间等预估变化）
5. 点击"生成修复配置"，切换 Nginx/Apache/Node.js 等平台查看配置代码
6. 点击"复测验证"，对比修复前后评分变化
7. 点击"下载 PDF 报告"导出完整安全报告

## 核心能力（已实现）

- 安全响应头扫描（HSTS / CSP / X-Frame-Options 等 15 个检测维度）
- SSL/TLS 证书检查（有效期、协议版本、密码套件）
- 敏感路径检测（.env / .git / /admin 等常见路径）
- 修复配置生成（Nginx / Apache / Express / Flask / Spring Boot / Cloudflare / Node.js / Python）
- 修复后复测（真实重新扫描，对比前后评分差异）
- PDF 报告导出（7 页专业报告）
- 批量扫描（最多 5 个 URL，asyncio 并发）
- AI 安全顾问（规则引擎 + 可选接入 OpenAI LLM）

## 安全边界

只做配置检测和探测性请求，不执行攻击利用；扫描前必须勾选授权确认；所有外部请求统一拦截内网地址（SSRF 防护）。
