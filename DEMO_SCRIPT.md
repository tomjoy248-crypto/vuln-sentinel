# 1 分钟演示脚本

## 开场（10秒）

"漏洞哨兵是一个面向中小团队的 Web 安全配置扫描与修复建议平台。传统安全扫描工具要么太贵，要么误报率太高。我们做了两个核心创新：

1. **多维交叉验证扫描** -- 用 5 维交叉验证，在当前基准测试集上实现 0% 误报
2. **扫描-修复-复测闭环** -- 发现漏洞后，生成修复配置，通过复测验证效果"

## 演示流程（50秒）

### 1. 首页 → 展示核心数据（10秒）
- 打开 https://vuln-sentinel-v11.onrender.com
- 指着 Dashboard 说："这是实时扫描数据，基于 OWASP Top 10，覆盖 15 个检测维度"
- 展示评分趋势图

### 2. 扫描演示（15秒）
- 输入 `https://example.com`，点击扫描
- 等待 3-5 秒出结果
- 指着报告说："当前评分，发现了若干安全问题，包括缺少 HSTS 和 CSP 等配置项"

### 3. 查看漏洞详情 + 修复建议（10秒）
- 展开第一个漏洞"缺少 HSTS"
- 展示：风险等级、OWASP 分类、修复代码、验证步骤
- "每个漏洞都有完整的修复方案，不只是告诉你有问题"

### 4. 生成修复配置（10秒）
- 点击"一键应用修复"查看修复配置
- 切换平台（Nginx / Apache / Node.js 等）查看对应配置
- "生成修复配置后可通过复测验证效果"

### 5. AI 顾问（5秒）
- 打开 AI 顾问，问"HSTS 是什么"
- 展示 AI 回复
- "AI 顾问能结合扫描结果，给出个性化的修复优先级"

## 结束语（5秒）

"整个流程：扫描 → 修复建议 → 复测对比。评委老师可以在 https://vuln-sentinel-v11.onrender.com 亲自体验，账号 demo / demo123"
