# 1 分钟演示脚本

## 开场（10秒）

"漏洞哨兵是一个面向中小团队的 Web 安全配置扫描与修复平台。传统安全扫描工具要么太贵，要么误报率太高。我们做了两个核心创新：

1. **零误报扫描**——用 5 维交叉验证，把误报率降到 0%
2. **一键修复闭环**——发现漏洞后，自动生成修复配置，验证修复效果"

## 演示流程（50秒）

### 1. 首页 → 展示核心数据（10秒）
- 打开 https://vuln-sentinel-v11.onrender.com
- 指着 Dashboard 说："这是实时扫描数据，基于 OWASP Top 10，覆盖 15 个检测维度"
- 展示评分趋势图

### 2. 扫描演示（15秒）
- 输入 `https://example.com`，点击扫描
- 等待 3-5 秒出结果
- 指着报告说："63 分，8 个发现，2 个高危——缺少 HSTS 和 CSP"

### 3. 查看漏洞详情 + 修复建议（10秒）
- 展开第一个漏洞"缺少 HSTS"
- 展示：风险等级、OWASP 分类、修复代码、验证步骤
- "每个漏洞都有完整的修复方案，不只是告诉你有问题"

### 4. 一键修复对比（10秒）
- 点击"一键应用修复"
- 展示修复前后对比：63 分 → 100 分
- "这是预估效果，实际应用到服务器后，可以重新扫描验证"

### 5. AI 顾问（5秒）
- 打开 AI 顾问，问"HSTS 是什么"
- 展示 AI 回复
- "AI 顾问能结合扫描结果，给出个性化的修复优先级"

## 结束语（5秒）

"整个闭环：扫描 → 发现 → 修复 → 验证。评委老师可以在 https://vuln-sentinel-v11.onrender.com 亲自体验，账号 demo / demo123"
